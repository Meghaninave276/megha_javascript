# 🌍 Country Explorer Web App

A responsive and interactive web application that allows users to explore countries, view their details, and save their favorite countries using local storage.

## 🚀 Features

🔍 Search Countries

Search for countries by name using a simple input field.

## 🌐 Display Country Data
Fetches real-time data from the REST Countries API including:
Country Name
Capital
Flag

## ⭐ Add to Favourites
Users can add countries to a favourites section.

## ❌ Remove from Favourites
Easily remove countries from the favourites list.

## 💾 Local Storage Support
Favourites persist even after refreshing the page.

## 🎨 Modern UI Design
Clean glassmorphism-style interface with responsive layout.

## 🛠️ Technologies Used
HTML5
CSS3 (Flexbox + Glassmorphism UI)
JavaScript (Vanilla JS)
Bootstrap 5
REST Countries API

⚙️ How It Works

On page load, the app fetches country data from:

https://restcountries.com/v3.1/all
All countries are displayed as cards.
Users can:
Search for a country
Add it to favourites
Remove it from favourites
Favourite countries are stored in localStorage and restored on reload.
